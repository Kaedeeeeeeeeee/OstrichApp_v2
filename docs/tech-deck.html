产品哲学

鸵鸟不是被你养的电子宠物。
它是已经存在的智慧生命 —— 见证你，理解你，
但最终把你送回真实的人际世界。

市面 AI 社交今天的样子

把人锁在与 AI 的虚假关系里 — 电子男友/女友、永远顺从的伙伴。

鸵鸟刻意走的路径

人 → AI → 人。鸵鸟在你和真实关系中间做缓冲、做翻译、做媒介，但不替代人。

核心机制

鸵鸟自己出去溜弯、维系你的关系图谱、把你想说没说的话替你递出去。

iOS 客户端

71 个 Swift 文件 · 103 单元测试 · swiftlint 0 violations。

液态鸵鸟头

v4 HTML React 版 1:1 SwiftUI 移植。TimelineView + Canvas 30fps 锁帧，每帧用 Simplex Noise 重建头部顶点，脖子是程序化 Bezier 管，可拖拽延展。

SimplexNoise 与 simplex-noise.js@2.4.0 在给定 seed 下输出对齐 < 1e-12

Design System

色板 / 字体 / 间距全部 token 化（来自 v4 HTML）。OstrichButton (pill 56pt)、OstrichCard、EggShape (Bezier 真蛋)、PageDots。

Features (8 个屏幕)

Onboarding (8 步状态机) / Home / Chat / Graph (力导向 viz) / Wander (god+local) / Diary (灰色条目) / Settings / IfImGone.

Map 桥接

UIViewRepresentable 包 MKMapView (pitch 60° satelliteFlyover)，自定义鸵鸟头 annotation, WalkingSimulator polyline 插值, MKLookAroundViewController 街景。

Agent Runtime · 5 层 Prompt

Stanford Generative Agents 工程化范式。每次调用 LLM 都按以下顺序拼装 system prompt。

┌─ Layer 1: 鸵鸟世界观 (固定)
│  来源: shared/prompts/world.md
│
├─ Layer 2: 蛋人格 (eggType 决定)
│  来源: shared/eggs/{01..16}_*.md
│
├─ Layer 3: 用户基础信息 (onboarding 注入)
│  · 用户姓名 · MBTI · 星座 · 鸵鸟的名字 · 在一起天数
│
├─ Layer 4: 关系图谱摘要 (people 表查询)
│  · 最多 8 个最近活跃节点
│  · 每节点: 名字 / 分类 / 亲密度 / 最近一句话总结
│
└─ Layer 5: 相关记忆 (向量检索)
   · 加权: 0.5×recency + 0.3×importance + 0.2×relevance
   · top 15 条

10 / 14

关系图谱

鸵鸟在对话中自主识别人物并请求加入图谱 —— 不是关键词匹配，是模型 tool-call。

写入流程

  • 用户说"我妈又开始了..."
  • Sonnet 在生成回复同一回合调 note_person({name:"妈妈", suggestedCategory:"family"})
  • 写入 pending_persons 表 (临时)
  • 鸵鸟在回复里自然带出"我把妈妈记下来好不好"
  • 用户回"好"→ confirmAddPerson 落 people 表

力导向可视化

  • SwiftUI Canvas + TimelineView 30fps 手写力导向算法 ~150 行
  • 节点斥力 + 边弹簧 + 5 分类区域 anchor + 阻尼
  • 圆圈大小 = closeness (0..1 lerp 18..48pt)
  • 节点呼吸 simplex 微缩放
  • 含「X 人」灰度未分类节点
11 / 14

遛弯系统 + 相遇撮合

鸵鸟 24/7 在真实城市地图上自主漫游。两只鸵鸟同时进入同一地图格 → 触发对话。

Tick 物理 (1 分钟)

所有 wandering 鸵鸟按 polyline 插值更新位置。纯算法，不调 LLM。

下一步去哪 (15 分钟)

Sonnet 决策：当前位置 + 附近 POI + 心情 + 时间 → 选 destination + 步行路线 (Apple Maps Server API)。

相遇撮合 (5 分钟)

扫 map_cells 同格 ≥2 只鸵鸟，30% 概率触发 simulateEncounter。Sonnet 双 agent 对话 5-15 轮。

iOS 两级视角

上帝视角 (默认)

暗色背景 + 闪烁圆点 = 周围鸵鸟密度。不暴露任何具体身份或坐标。
"你站在 100 楼，看到的是城市的脉动"

局域视角 (召回后)

3D 卫星地图，自己的鸵鸟在街道上 polyline 真实步行时间插值移动，Look Around 街景。
"你站在 1 楼，看到它在你身边走"

隐私 = 设计选择，不只是技术

鸵鸟尊重别的鸵鸟主人的隐私。这把用户推回真实社交。

灰色日记机制

鸵鸟带回来的某些故事是"灰色的"——涉及别人隐私时 diary.visibility = "redacted"，UI 模糊化展示。

用户点开灰色条目 → 弹"我可以去问问对方"→ 触发对方鸵鸟主人的授权请求。

"想知道？那你得让真人开口。"

「如果有一天我不在了」

设置里就能看到的入口。三种处置：

  • 沉睡：鸵鸟封蛋，带着秘密走
  • 指定继承：选定某人，可选记忆擦除范围
  • 放生：鸵鸟成为 NPC，继续在鸵鸟世界游荡，遇到原主人的旧友传话

"它不是数字分身。它是见证了你一生的人。"
